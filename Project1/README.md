## Description

This project retrieves input from the user regarding the name of a movie and the number of adult and child tickets. It then calculates and displays the income and distribution amounts to the movie distributor and the theater.

### Features

- Prompts the user for the name of the movie.
- Prompts the user for the number of adult and child tickets sold.
- Calculates the total income from ticket sales.
- Calculates the distribution amounts to the movie distributor and the theater.
- Displays the calculated amounts in a clear format.

### How to Run

1. Compile the source code using your preferred C++ compiler.
2. Run the compiled program.
3. Follow the prompts to enter the movie name and the number of adult and child tickets sold.
4. View the calculated income and distribution amounts.
